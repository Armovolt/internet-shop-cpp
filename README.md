# Інформаційна система «Облік Продажів»

Консольна утиліта для автоматизації обліку номенклатури товарів, ведення бази контрагентів та фіксації угод. 
Виконано в рамках навчальної практики. Спеціальність 121 «Інженерія програмного забезпечення».

## Опис задачі

Утиліта дозволяє користувачу керувати каталогом товарів, базою покупців та оформлювати угоди продажу. Усі дані зберігаються локально у вигляді файлів та доступні між сесіями запуску програми. Програма реалізована на стандартному C++ без використання сторонніх бібліотек, що гарантує її кросплатформеність та 100% сумісність із будь-яким середовищем розробки (наприклад, Visual Studio Code).

## Структура бази даних

Система складається з трьох взаємопов'язаних таблиць.

### 1. Товари (Каталог)
Основна таблиця для ведення каталогу. Включає цінову політику та опис номенклатури.
* **Код товару:** Унікальний ідентифікатор (PK)
* **Найменування:** Назва продукту
* **Гуртова ціна:** Ціна для великих партій
* **Роздрібна ціна:** Ціна для кінцевого споживача
* **Опис:** Характеристики товару

### 2. Покупці (База контактів)
Таблиця для зберігання інформації про клієнтів.
* **Код покупця:** Унікальний ідентифікатор (PK)
* **Телефон:** Контактний номер
* **Адреса:** Адреса доставки/реєстрації
* **Контактна особа:** ПІБ покупця

### 3. Угоди (Продажі)
Сполучна ланка, що забезпечує цілісність зв'язків між асортиментом та клієнтською базою.
* **Transaction_ID:** Унікальний код угоди
* **Дата:** Дата та час здійснення купівлі
* **Код товару:** Посилання на куплений товар (FK)
* **Кількість:** Кількість купленого товару
* **Код покупця:** Посилання на покупця (FK)
* **Ознака гурту:** Ознака оптового продажу (1 — опт, 0 — роздріб)

## Функціонал програми

| № | Модуль | Доступні дії |
|---|--------|--------------|
| 1 | Товари | Додавання, перегляд каталогу, збереження (TXT/BIN), завантаження з BIN |
| 2 | Покупці| Реєстрація клієнта, перегляд бази, збереження (TXT/BIN), завантаження з BIN |
| 3 | Угоди  | Оформлення продажу, перегляд історії, збереження (TXT/BIN), завантаження з BIN |
