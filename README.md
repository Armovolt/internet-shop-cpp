# Інформаційна система «Інтернет-магазин»

Консольна утиліта для автоматизації роботи комерційного відділу Інтернет-магазину: ведення каталогу товарів, обліку клієнтів та фіксації продажів. 
Виконано в рамках навчальної практики. Спеціальність 121 «Інженерія програмного забезпечення».

## Опис задачі
Програма дозволяє керувати базою даних Інтернет-магазину. **Автоматизовано систему лояльності:** при здійсненні покупок на загальну суму понад 5000 грн клієнт автоматично переходить у категорію постійних (VIP), що надає йому знижку 2% на всі подальші угоди. Всі таблиці оптимізовані під стандартний вивід у консоль шириною рівно 80 символів. Дані зберігаються локально у текстових та бінарних файлах.

## Структура бази даних

Система складається з трьох взаємопов'язаних таблиць:

### 1. Товари
* **Код товару** (PK)
* **Назва** (Найменування продукту)
* **Ціна** (Вартість одиниці)
* **Одиниця виміру** (штуки, кг, літри тощо)

### 2. Клієнти
* **Код клієнта** (PK)
* **Прізвище, Ім'я, По батькові** (Анкетні дані)
* **Адреса**
* **Телефон**
* **e-mail**
* **Ознака постійного клієнта** (1 - VIP, 0 - Звичайний)

### 3. Продажі
* **Код продажу** (PK)
* **Код товару** (FK)
* **Код клієнта** (FK)
* **Дата продажу**
* **Дата доставки**
* **Кількість**

## Функціонал програми

| № | Модуль | Доступні дії |
|---|--------|--------------|
| 1 | Товари | Додавання товарів, перегляд каталогу, експорт у TXT, збереження/завантаження (BIN) |
| 2 | Клієнти| Реєстрація клієнтів, перегляд контактів, експорт у TXT, збереження/завантаження (BIN) |
| 3 | Продажі| Оформлення угод (з розрахунком знижок), перегляд історії, експорт у TXT, збереження/завантаження (BIN) |

## Особливості реалізації
* **Захист від некоректного вводу:** Реалізовано функції безпечного зчитування (`SafeInputInt`, `SafeInputFloat`). Введення літер замість чисел не призводить до аварійного завершення утиліти.
* **Бізнес-логіка:** Програма самостійно розраховує суму до оплати, перевіряє статус клієнта, застосовує знижки та оновлює статус клієнта у базі при досягненні ліміту покупок у 5000 грн.
* **Кросплатформеність:** Код написано на стандартному C++ (без прив'язки до ОС чи специфічних бібліотек), що гарантує 100% сумісність із будь-яким компілятором чи середовищем розробки (наприклад, Visual Studio Code).
