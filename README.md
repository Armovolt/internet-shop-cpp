# Інформаційна система «Інтернет-магазин»

Консольна утиліта для автоматизації роботи комерційного відділу Інтернет-магазину: ведення каталогу товарів, обліку клієнтів та фіксації продажів. 
Виконано в рамках навчальної практики. Спеціальність 121 «Інженерія програмного забезпечення».

## Опис задачі
Програма дозволяє керувати базою даних Інтернет-магазину. **Автоматизовано систему лояльності:** при здійсненні покупок на загальну суму понад 5000 грн клієнт автоматично переходить у категорію постійних (VIP), що надає йому знижку 2% на всі подальші угоди. Всі таблиці оптимізовані під стандартний вивід у консоль шириною рівно 80 символів завдяки окремим функціям формування шапки таблиць. Дані зберігаються локально у текстових та бінарних файлах.

## Як запустити
Проєкт написаний на стандартному C++ і не потребує встановлення жодних зовнішніх бібліотек.
1. Завантажте файли коду в одну папку.
2. Відкрийте папку в будь-якому середовищі розробки (наприклад, **Visual Studio Code** або **OnlineGDB**).
3. Скомпілюйте та запустіть програму.

## Структура бази даних

Система складається з трьох взаємопов'язаних таблиць:

### 1. Товари
* **Код товару** (PK)
* **Назва** (Найменування продукту)
* **Ціна** (Вартість одиниці)
* **Одиниця виміру** (штуки, кг, літри тощо)

### 2. Клієнти
* **Код клієнта** (PK)
* **Прізвище, Ім'я, По батькові** (Анкетні дані)
* **Адреса**
* **Телефон**
* **e-mail**
* **Ознака постійного клієнта** (1 - VIP, 0 - Звичайний)

### 3. Продажі
* **Код продажу** (PK)
* **Код товару** (FK)
* **Код клієнта** (FK)
* **Дата продажу**
* **Дата доставки**
* **Кількість**

## Функціонал програми

| № | Модуль | Доступні дії |
|---|--------|--------------|
| 1 | Товари | Додавання, перегляд каталогу, **редагування даних**, **сортування за кодом**, експорт у TXT, збереження/завантаження (BIN) |
| 2 | Клієнти| Реєстрація клієнтів, перегляд бази, **редагування даних**, **сортування за кодом**, експорт у TXT, збереження/завантаження (BIN) |
| 3 | Продажі| Оформлення угод (зі знижками), перегляд історії, **редагування даних**, **сортування за кодом**, експорт у TXT, збереження/завантаження (BIN) |

## Особливості реалізації
* **Захист від некоректного вводу:** Реалізовано функції безпечного зчитування (`SafeInputInt`, `SafeInputFloat`). Введення літер замість чисел не призводить до аварійного завершення утиліти.
* **Управління даними:** Реалізовано модулі сортування масивів структур за зростанням ключових полів (ID) та можливість динамічного редагування будь-якого запису користувачем.
* **Форматування виводу:** Використано спеціалізовані функції формування шапок таблиць (`ShapkaProducts`, `ShapkaClients`, `ShapkaSales`) для забезпечення фіксованої ширини виводу рівно в 80 символів із коректною обробкою символів кирилиці.
* **Бізнес-логіка:** Програма самостійно розраховує суму до оплати, перевіряє статус клієнта, застосовує знижки та оновлює статус клієнта у базі при досягненні ліміту покупок у 5000 грн.
